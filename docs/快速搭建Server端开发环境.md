快速搭建Server端开发环境
===


安装
---

按照您的操作系统分别安装下列软件：

* [git客户端][10]或者github的客户端（[MAC版][20]，[Windows版][25]，[github官方文档][27]）
* [Node.js][30]
* [Apache CouchDB][40]
* 你喜欢的文本编辑器（可选），推荐使用[Sublime Text][50]


Checkout代码
---
在命令行窗口运行：

	> git clone https://github.com/dewafer/next.git
	
或者使用github的客户端来clone。方法是安装github客户端后点击本项目首页右下角的`Clone in Desktop`按钮。
